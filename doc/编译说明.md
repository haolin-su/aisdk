# 编译说明

## 前提条件

- 系统：Ubuntu 16.04 或更高版本
- 软件：
    - Go 1.13 或更高版本
    - Node.js 10 或更高版本
    - Yarn 1.16 或更高版本

## 编译步骤

1. 安装 Go 语言环境：

```
sudo apt-get update
sudo apt-get install golang-1.13-go
```

2. 配置 Go 语言环境变量：

```
export GOROOT=/usr/lib/go-1.13
export GOPATH=$HOME/go
export PATH=$PATH:$GOROOT/bin:$GOPATH/bin
```

3. 安装 Node.js 环境：

```
sudo apt-get install nodejs
```

4. 安装 Yarn 包管理器：

```
sudo npm install -g yarn
```
